预置三个演示场景

🛒 场景一：标准订单流程

用户下单 → 支付 → 商家发货 → 用户收货 → 完成

5个步骤 · 约2分钟演示

↩️ 场景二：退款审批流程

用户申请退款 → 客服审核 → 经理审批 → 退款到账

4个步骤 · 约1.5分钟演示

📦 场景三：商品上架审批

运营提交商品 → 管理员审核 → 商品上架 → 用户可见

4个步骤 · 约1.5分钟演示

交互逻辑

  • 场景切换：顶部下拉框切换三个预置场景，自动重置进度
  • 分步推进：每次点击"下一步"，触发动画过渡（节点高亮 → 进度条推进 → 详情刷新）
  • 自动演示：可选加一个"自动播放"按钮（每步间隔2秒），留到二期
  • 数据隔离：演示数据全部带 DEMO 前缀，独立于正式数据，可随时重置
